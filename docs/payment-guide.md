## License Key

### Format
License key menggunakan format: `XXXX-XXXX-XXXX-XXXX` (16 karakter, uppercase)

### Aktivasi

1. Install plugin WordPress
2. Buka **AI Generator > Settings**
3. Masukkan license key di kolom "License Key"
4. Klik "Validate"
5. Jika valid, akan muncul notifikasi sukses

### Deaktivasi

License key dapat ditransfer ke domain lain dengan:
1. Deaktivasi di domain lama (Settings > deactivate)
2. Aktivasi baru di domain tujuan

**Batasan:** 1 license key berlaku untuk 1 domain.
